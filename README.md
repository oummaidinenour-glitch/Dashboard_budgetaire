## 🎯 Objectif du Projet

Analyser la **performance financière** d'une organisation en comparant le **budget prévisionnel** avec les **dépenses réellement effectuées**, afin de :
- Identifier rapidement les **écarts budgétaires**
- Faciliter la **prise de décision stratégique**
- Visualiser les **tendances mensuelles** par département

---

## 📂 Structure du Fichier Excel

Le fichier Excel contient **4 feuilles** :

| Feuille | Contenu |
|---------|---------|
| 🎯 Dashboard | Tableau de bord principal avec KPIs et graphiques |
| 📊 Données Brutes | Les 480 lignes de données sources |
| 📋 Analyse Départements | Comparaison par département |
| 📈 Analyse Postes & Alertes | Analyse des postes de charges et alertes |

## 📈 Description des Données

- 🗂️ **480 lignes** de données
- 🏢 **5 départements**
- 💰 **8 postes de charges**
- 📅 **12 mois** d'activité (année complète)

---

## 🛠️ Méthodologie

1. **Nettoyage et préparation** des données brutes
2. **Création des KPIs** : Budget total, Réalisé, Écart budgétaire
3. **Conception de visualisations interactives** pour une analyse claire et rapide
